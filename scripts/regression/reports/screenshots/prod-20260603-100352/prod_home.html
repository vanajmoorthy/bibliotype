How to Get Your File

Desktop

  1. Go to https://www.goodreads.com/review/import.
  2. Click the "Export Library" button and wait for the file to generate.
  3. Click the link to download the goodreads_library_export.csv file.
  4. Drop the file in the upload box.

Mobile

  1. Go to https://www.goodreads.com/review/import.
  2. If it opens on web and you're already logged in, follow the instructions above.
  3. If it opens in the app, click the "Export Library" button and wait for the file to generate.
  4. Click the link, then click the share button (Android , iOS ) in the bottom right
  5. Click copy, then go to your browser and paste in the link.
  6. You will need to log in to Goodreads again if you're not already.
  7. This will download the goodreads_library_export.csv file, come back to Bibliotype and upload the file here.
