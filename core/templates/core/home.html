How to Get Your File

Goodreads

  1. Go to https://www.goodreads.com/review/import.
  2. Click the "Export Library" button and wait for the file to generate.
  3. Download the `goodreads_library_export.csv` file.

StoryGraph

  1. Navigate to your main profile page.
  2. Click "Manage library", then choose "Import/Export".
  3. Under the "Export Library" section, click the "Export" button.
  4. Download the `.csv` file (e.g., `storygraph-export.csv`).
